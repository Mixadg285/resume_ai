# Welcome to AI Resume Checker
This web application enables user to validate their resume using AI by give scores and some feedbacks

# React + Vite
 
1. Start the app (cd resume)

   ```bash
   npx expo start 
   ```
2. run the backend (cd backend)

   ```bash
   node server.js 
   ```
